# Designing a Reading Ritual

A personal blog should feel easy to enter. The reader needs a clear title, a calm line length, enough contrast, and a sense that the page was made with care.

## A few useful choices

- Keep the post list visual, but let the writing stay in charge
- Use tags for navigation instead of adding too many menu links
- Give each post a strong cover image
- Keep article pages narrow enough for focused reading

## Markdown that stays friendly

Write naturally. Use headings when the thought changes. Use links when they help, like the [GitHub Pages documentation](https://docs.github.com/pages), and use code when precision matters.

```html
<article>
  <h1>Your note title</h1>
  <p>Your careful paragraph.</p>
</article>
```

The page should get out of the way while still feeling considered.
